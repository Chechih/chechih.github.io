在 VirtualBox 中安裝 Ubuntu

📁 VirtualBox Ubuntu
七月 20, 2024

前置準備

  • 前往 VirtualBox 官網 下載 EXE 安裝，下載完成安裝起來
  • 前往 Ubuntu 官網 下載印象檔
    • 此範例使用 24.04 LTS 版本

開始安裝

Step1: 開啟安裝好的 VirtualBox

Step2: 點選右上角的新增按鈕

Step3: 輸入參數，以下是說明
a. 名稱: 打 Ubuntu 即可，這是之後用來是別虛機的名稱
b. 資料夾: 是建立起的虛機檔案存放路徑，如果沒有特別需要預設即可
c. 印象檔: 使用剛剛 Ubuntu 下載下來的 ISO 檔案
d. 略個無人職守安裝: 有點像之後自動幫你安裝到系統起來，不過他要填的訊息也比較多，先勾起來，用一般安裝

Step4: 設定記憶體和 CPU 大小，一需求設定

Step5: 設定虛擬硬碟，這個有點像系統要跑在多大的硬碟上
選項 1. 建立一個，但就算你輸入 100G 也不會直接吃掉你硬碟的 100G，除非你勾選的預設配置完整大小
選項 2. 指的是使用已建立好的
選項 3. 先不處理，之後也可以到設定那邊處理
建議如果第一次用的話，先選擇 1.

Strp6: 確認剛剛輸入的值，沒問題直接按完成


Step7: 先別急著啟動，先進行初始設定，點選剛剛建立的虛機，再按設定

點選一般>進階將共用剪貼簿和拖拉設定成雙向，這能讓之後複製和拖拉更方便。

點選儲存裝置>控制器，勾選固態硬碟

點選網路增加新的網卡2，僅限主機，顧名思義就是宿主能連近來而已，網卡2，選擇橋接介面卡，這能把虛機放到內網裡，其他電腦能 Ping 的到


備註: 共用資料夾，就是讓宿主和虛機能共用的資料夾，之後有需要再設定即可
沒問題先按確定

共享設置

Step 1: 插入 Guest Additions CD

Step 2: 打開終端機

Step 3: 安裝 Guest Additions CD，輸入

sudo ./VBoxLinuxAdditions.run


Step 4: 發現他還要安裝 bzip2，那先安裝起來

sudo apt-get install bzip2


Step 5: 安裝完後再是一次，這次應該可以順利安裝了

sudo ./VBoxLinuxAdditions.run


Step 6: 重新開機，開完機應該可以發現，視窗可以依比例伸縮

reboot

Step 7: 設定共用剪貼簿和共用資料夾，如果前面有設定這裡可以跳過，共用資料夾一樣需要再來設定也可以

一般套件安裝和更新

  • 更新
sudo apt-get update
sudo apt-get upgrade
  • 安裝 vim
sudo apt-get install -y vim
  • 安裝 openssh server
sudo apt-get install -y openssh-server
  • 安裝 net tools
sudo apt-get install net-tools
