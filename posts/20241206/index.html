Docker Desktop 安裝

📁 Docker
十二月 6, 2024

安裝環境

Windows 2022

前置作業

到空置台開起 Hyper-V

下載 Docker Desktop

到 Docker 官網 下載安裝
下載完成

安裝

  1. 開啟執行檔
  2. 勾選設定，第一個選項是指要不要使用 Hyper-V 安裝，若沒有特別需求，使用預設即可
  3. 開始安裝，讓他跑一下子
  4. 安裝完成，接著按下去，電腦會重新開機
  5. 開機好後，他會跳出簽署強迫同意書
  6. 再來他會跳出設定，預設即可
  7. 這邊都可以直接跳掉，不輸入
  8. 基本上這樣就可以使用了

以前安裝發生過的錯誤

狀況 1:

請到 cmd 輸入

wsl --update

安裝完成後 重新開機 狀況 2:

這是因為 Hyper-V 沒有開

確認安裝

之後在 cmd 輸入以下指令確認一下

docker --version

而輸入

docker version

可以看到更詳細的訊息
